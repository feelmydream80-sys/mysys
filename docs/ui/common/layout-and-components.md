# 공통 UI 레이아웃 및 컴포넌트 (Common Layout & Components)

**Cline 필수 읽기 문서** — 변경 빈도가 낮은 공통 가이드

## 1. 기본 레이아웃
- `templates/base.html` 상속 구조
- 공통 요소: Navbar, Sidebar, Footer, Flash Message 등

## 2. 재사용 가능한 컴포넌트
- 테이블, 카드, 모달, 폼 그룹 등
- 위치: `templates/components/`

## 3. UI 디자인 원칙
- 일관된 스타일링 (Bootstrap / Tailwind 등)
- 반응형 디자인
- 접근성 고려

## 4. 게시판 설계 원칙
- 일관된 스타일링 (Bootstrap / Tailwind 등)
- 반응형 디자인
- 접근성 고려
- 페이징 기능 및 표출 수량 콤보박스 기능 삽입
- 


**Cline 지침**: 모든 화면은 이 공통 가이드를 준수해야 하며, 새로운 컴포넌트는 여기에 기록한다.
