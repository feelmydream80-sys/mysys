화면 관련 작업 시 **반드시 이 파일을 최우선으로 읽고 숙지**하세요.

## Cline 작업 지침 (매우 중요)

- 화면(UI) 관련 요구사항이 들어오면 **반드시 이 파일을 먼저 읽고** 기존 화면 도메인과 비교 검토한다.
- 기존 화면과 유사한 기능이면 재사용성을 고려한다.
- 새로운 화면을 구현할 때는 이 파일에 먼저 해당 화면 정의를 추가하거나 업데이트한 후 구현을 시작한다.
- 화면 구현 내용과 이 문서가 불일치할 경우, 즉시 사용자에게 검토 결과를 제시하고 확인을 받는다.
- 변경이 잦은 부분이므로, 구현 완료 후 **반드시 이 파일을 최신 상태로 유지**한다.
