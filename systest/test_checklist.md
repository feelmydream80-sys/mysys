## 합/불 판정 기준

### 1. 페이지 로딩
- **합격**: ≤ 5초
- **불합격**: > 5초

### 2. API 호출
- **합격**: 응답 코드 200, 데이터 구조 정확
- **불합격**: 응답 코드 4xx/5xx, 데이터 누락/오류

### 3. UI 상호작용
- **합격**: 버튼 클릭 → 이벤트 발생 → 요청 전송 → 응답 수신 → UI 업데이트
- **불합격**: 버튼 클릭 → 이벤트 미발생, 요청 오류, 응답 오류, UI 미업데이트

### 4. 데이터 처리
- **합격**: 데이터 추가/수정/삭제 → DB 반영 → UI 갱신
- **불합격**: 데이터 처리 중 오류, DB 미반영, UI 갱신 오류

### 5. 오류 처리
- **합격**: 오류 발생 → 사용자 메시지 표시 → 오류 로그 기록
- **불합격**: 오류 발생 → 비정상 종료, 메시지 미표시, 정보 누출
