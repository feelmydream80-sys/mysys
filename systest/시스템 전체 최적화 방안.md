# 시스템 전체 최적화 방안

## 1. 개요
MSYS 시스템은 Flask 기반의 데이터 수집·분석 플랫폼으로, 관리자 설정, 대시보드, 데이터 분석, 사용자 관리 등의 기능을 제공합니다. 현재 시스템은 여러 라우트·서비스·DAO로 구성되어 있으나, 중복 코드, 의존성 관리 문제, 테스트ability 저하 등의 문제가 존재합니다. 본 문서에서는 전체 시스템의 최적화 방안을 제시합니다.

## 2. 전체 시스템 구조 분석

### 2.1 라우트/API 계층
- **auth_routes.py**: 로그인, 회원가입, 권한 관리 등의 사용자 인증 기능
- **mngr_sett_routes.py**: 관리자 설정(사용자 권한, 아이콘, 스케줄 설정)
- **analysis_routes.py**: 데이터 분석 차트·보고서 기능
- **api/*.py**: RESTful API 엔드포인트 (대시보드, 분석, 데이터 정의 등)
- **ui/*.py**: 사용자 인터페이스 라우트 (대시보드, 수집 일정 등)

### 2.2 서비스 계층
- **user_service.py**: 사용자 정보 조회·수정·삭제
- **dashboard_service.py**: 대시보드 데이터 조회·처리
- **mngr_sett_service.py**: 관리자 설정 로직
- **analysis_service.py**: 데이터 분석 로직
- **auth_service.py**: 인증·권한 관리 로직

### 2.3 DAO/매퍼 계층
- **dao/*.py**: 데이터베이스 접근 객체
- **mapper/*.py**: 데이터베이스 결과 매핑

### 2.4 유틸리티 계층
- **utils/logging_config.py**: 로깅 설정
- **utils/datetime_utils.py**: 날짜·시간 처리
- **utils/auth_middleware.py**: 인증 미들웨어
- **msys/database.py**: 데이터베이스 연결 관리

## 3. 최적화가 필요한 문제점 분류

### 3.1 공통 모듈화 필요성
1. **데코레이터 중복**: `@login_required`, `@check_password_change_required`, `@admin_required` 등이 모든 API에 중복 적용
2. **로깅 패턴 중복**: 각 서비스·라우트에서 비슷한 로깅 패턴이 반복
3. **DB 연결 관리**: 각 API에서 직접 `get_db_connection()`을 호출하고 트랜잭션 관리

### 3.2 의존성 관리 문제
1. **서비스 계층의 순환 의존**: `MngrSettService.__init__()`에서 로컬 import로 순환 의존 해결
2. **하드코딩 의존**: `DEFAULT_ADMIN_SETTINGS`가 서비스 내에 직접 정의
3. **테스트ability 저하**: 서비스 클래스가 DB 연결에 직접 의존

### 3.3 중복 코드 문제
1. **사용자 권한 업데이트**: `mngr_sett_routes.py`에서 `save_all_settings()`와 `update_bulk_user_permissions()`에 중복 로직
2. **데이터 조회 로직**: 여러 API에서 같은 쿼리를 반복 호출

### 3.4 코드 품질 문제
1. **비효율한 쿼리**: `get_all_mngr_sett()`에서 여러 서비스를 호출하며 중복 쿼리 발생
2. **잠재적 오류**: `reload_mappings()`가 각 서비스에서 호출되어 성능 저하
3. **문서화 부족**: 일부 API에 Swagger 문서가 부족

## 5. 구체적인 실행 계획

### 5.1 단계 1: 공통 모듈화

#### 5.1.1 데코레이터 통합
```python
# utils/decorators.py
from functools import wraps
from flask import session, jsonify, render_template, request
from models.user import User

def login_required(f):
    @wraps(f)
    def decorated_function(*args, **kwargs):
        if 'user' not in session:
            return jsonify({"error": "로그인이 필요합니다."}), 401
        return f(*args, **kwargs)
    return decorated_function

def check_password_change_required(f):
    @wraps(f)
    def decorated_function(*args, **kwargs):
        if request.endpoint in ['auth.change_password', 'auth.logout']:
            return f(*args, **kwargs)
        
        is_admin = 'mngr_sett' in session.get('user', {}).get('permissions', [])
        if is_admin:
            return f(*args, **kwargs)
        
        if session.get('force_password_change'):
            return jsonify({"error": "비밀번호를 변경해야 합니다."}), 403
        return f(*args, **kwargs)
    return decorated_function

def permission_required(permission):
    def decorator(f):
        @wraps(f)
        def decorated_function(*args, **kwargs):
            user_permissions = session.get('user', {}).get('permissions', [])
            if permission not in user_permissions:
                return jsonify({"error": "권한이 필요합니다."}), 403
            return f(*args, **kwargs)
        return decorated_function
    return decorator
```

#### 5.1.2 DB 연결 관리
```python
# utils/db.py
from contextlib import contextmanager
from msys.database import get_db_connection

@contextmanager
def db_connection():
    conn = None
    try:
        conn = get_db_connection()
        yield conn
        conn.commit()
    except Exception as e:
        if conn:
            conn.rollback()
        raise
    finally:
        if conn:
            conn.close()
```

#### 5.1.3 로깅 유틸리티
```python
# utils/logging_config.py
def log_api_request(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        logger.info(f"=== API: {func.__name__} 시작 ===")
        try:
            result = func(*args, **kwargs)
            logger.info(f"=== API: {func.__name__} 완료 ===")
            return result
        except Exception as e:
            logger.error(f"❌ API: {func.__name__} 실패: {e}", exc_info=True)
            raise
    return wrapper
```

```python
# service/user_service.py
def update_user_permissions(self, user_id, menu_ids):
    """사용자의 메뉴 접근 권한을 업데이트합니다. (단일/대량 업데이트 지원)"""
    try:
        self.user_mapper.update_user_permissions(user_id, menu_ids)
        
        log_message = f"Permissions for user '{user_id}' updated. New permissions: {menu_ids}"
        self._log_user_event(user_id, 'AUTH_UPDATE_PERM', log_message)
    except Exception as e:
        raise
```

```python
# service/mngr_sett_service.py
def get_all_settings(self) -> list[dict]:
    """모든 관리자 설정을 조회합니다. (중복 쿼리 제거)"""
    self.logger.info("=== Service: get_all_settings() 시작 ===")
    try:
        mapper = self.mngr_sett_mapper
        all_settings_raw = mapper.get_all_settings()
        all_icons = self.icon_service.get_all_icons_data()
        
        # con_hist_summary는 필요한 경우만 가져오기
        if need_con_hist_summary:
            con_hist_summary = self.dashboard_service.get_summary(all_data=True)
        else:
            con_hist_summary = []
        
        result = self._combine_settings_details(all_settings_raw, con_hist_summary, all_icons)
        return result
    except Exception as e:
        self.logger.error(f"Service: Failed to get all settings: {e}", exc_info=True)
        return []
```
